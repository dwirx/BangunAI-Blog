## Rolling Release

Tidak perlu upgrade besar setiap 6 bulan. Semua update datang terus-menerus.

## AUR (Arch User Repository)

Hampir semua software ada di AUR. Install tinggal `yay -S nama-package`.

## Minimalis by Default

Arch tidak install apa-apa yang tidak kamu minta. Kamu mulai dari nol dan membangun sistem sesuai kebutuhan.

> "I don't use Arch because it's hard. I use it because once you set it up, everything just makes sense."

## Arch Wiki

Referensi Linux terlengkap yang pernah ada. Hampir semua masalah Linux bisa dijawab di sana.

## Kesimpulan

Arch bukan untuk semua orang. Tapi kalau kamu suka kontrol penuh, Arch sangat rewarding.
