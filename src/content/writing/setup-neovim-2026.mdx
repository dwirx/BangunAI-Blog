## Filosofi Setup

Prinsip saya: **seminimal mungkin, se-powerful mungkin**. Setiap plugin harus punya alasan jelas.

## Plugin Inti

```lua
require("lazy").setup({
  "nvim-treesitter/nvim-treesitter",
  "neovim/nvim-lspconfig",
  "hrsh7th/nvim-cmp",
  "nvim-telescope/telescope.nvim",
  "lewis6991/gitsigns.nvim",
})
```

## Keybindings Favorit

- `<leader>ff` — find files
- `<leader>fg` — live grep
- `gd` — go to definition
- `K` — hover docs

## Kesimpulan

Neovim bukan untuk semua orang, tapi kalau sudah "klik", susah balik ke editor lain.
